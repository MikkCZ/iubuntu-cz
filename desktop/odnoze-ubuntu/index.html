Odnože Ubuntu

Odnože Ubuntu nabízejí unikátní způsob práce s Ubuntu, každá z nich se svou osobitou tváří a různými výchozími aplikacemi a nastaveními. Uživatelé těchto verzí mají přístup do plného archivu Ubuntu pro instalaci aplikací a aktualizace.

  • Kubuntu

    Kubuntu nabízí pracovní plochu KDE Plasma a dobře vypadající systém pro domácí a kancelářské použití.

  • Lubuntu

    Lubuntu je rychlá, energeticky šetrná a lehká varianta Ubuntu používající prostředí LXQt. Je oblíbenou odnoží pro uživatele se slabšími počítači a notebooky.

  • Ubuntu Budgie

    Remix Ubuntu s univerzálním na distribuci nezávislým prostředím, využívající moderní technologie GNOME.

  • Ubuntu Kylin

    Projekt Kylin je vyladěn potřebám Čínských uživatelů, poskytující promyšlený a elegantní zážitek přímo od základu.

  • Ubuntu MATE

    Ubuntu MATE dává na odiv jednoduchost klasického prostředí. MATE je stále vyvíjeným pokračovatelem prostředí GNOME 2, které používalo i standardní Ubuntu až do října 2010.

  • Xubuntu

    Xubuntu je elegantní a jednoduše použitelný operační systém. Je postaveno na prostředí Xfce, které je stabilním, lehkým a velmi dobře konfigurovatelným rozhraním.

  • Ubuntu Studio

    Ubuntu Studio je odnož Ubuntu zaměřená na tvůrce multimediálního obsahu, vhodná pro audio, video a grafické nadšence a profesionály.
